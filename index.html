01Basics

Order of a matrix

A matrix with m rows and n columns has order m x n.

  • CBSE writing: Write order before doing operations.
  • JEE focus: Dimension check eliminates many wrong options quickly.
  • Common trap: Do not confuse m x n with n x m.
02Types

Square and diagonal matrices

A square matrix has same number of rows and columns; a diagonal matrix has non-zero entries only on the main diagonal.

  • CBSE writing: State the condition on non-diagonal entries.
  • JEE focus: Diagonal matrices simplify powers and products.
  • Common trap: Every diagonal matrix is square, but every square matrix is not diagonal.
03Types

Identity matrix

Identity matrix I has 1 on the main diagonal and 0 elsewhere.

  • CBSE writing: Use AI=IA=A when dimensions match.
  • JEE focus: Identity behaves like number 1 in matrix multiplication.
  • Common trap: The order of I must match the multiplication.
04Operations

Matrix addition

Matrices can be added only when their orders are the same.

  • CBSE writing: Add corresponding entries.
  • JEE focus: Use equality of matrices entry-wise.
  • Common trap: Matrices of different order cannot be added.
05Operations

Matrix multiplication

AB is defined only when columns of A equal rows of B.

  • CBSE writing: Show row by column multiplication clearly.
  • JEE focus: AB and BA may both exist but need not be equal.
  • Common trap: Matrix multiplication is not generally commutative.
06Transpose

Transpose properties

Transpose changes rows into columns and reverses product order.

  • CBSE writing: Use (AB)^T = B^T A^T.
  • JEE focus: Transpose identities simplify symmetric matrix questions.
  • Common trap: Do not write (AB)^T = A^T B^T.
07Special

Symmetric and skew-symmetric

A is symmetric if A^T=A and skew-symmetric if A^T=-A.

  • CBSE writing: For skew-symmetric matrices, diagonal entries are zero.
  • JEE focus: Every square matrix A can be split into symmetric and skew-symmetric parts.
  • Common trap: Skew-symmetric applies only to square matrices.
08Inverse

Inverse of a matrix

A square matrix A has inverse A^-1 if AA^-1=A^-1A=I.

  • CBSE writing: State non-singular condition before using inverse.
  • JEE focus: Inverse helps solve AX=B as X=A^-1B.
  • Common trap: Only square non-singular matrices have inverse.

Formula sheet

High-yield formulas with exact usage conditions.

Use these as quick revision cards before solving examples.

Order

Order rule

A = [a_ij] has order m x n when 1 <= i <= m and 1 <= j <= n

Rows are counted first, columns second.

  • Example: A 2 x 3 matrix has 6 entries.
  • Trap: Do not reverse row and column count.
Addition

Addition rule

(A+B)_ij = a_ij + b_ij

Only same-order matrices can be added.

  • Example: [[1,2],[3,4]] + [[2,0],[1,3]] = [[3,2],[4,7]].
  • Trap: Different orders mean addition is undefined.
Scalar

Scalar multiplication

(kA)_ij = k a_ij

Every entry is multiplied by k.

  • Example: 2[[1,-1],[0,3]] = [[2,-2],[0,6]].
  • Trap: Do not multiply only diagonal entries.
Product

Matrix product

(AB)_ij = sum a_ik b_kj

Use row i of A and column j of B.

  • Example: For 2 x 2 matrices, first entry is a11b11+a12b21.
  • Trap: Order matters.
Transpose

Transpose of sum

(A+B)^T = A^T + B^T

Transpose distributes over addition.

  • Example: Transpose after adding or add after transposing.
  • Trap: Orders must match.
Transpose

Transpose of product

(AB)^T = B^T A^T

Product order reverses.

  • Example: If AB exists, transpose product by reversing.
  • Trap: Most common error is keeping same order.
Split

Symmetric split

A = (A+A^T)/2 + (A-A^T)/2

First part is symmetric and second is skew-symmetric.

  • Example: Useful for proving decomposition.
  • Trap: Works only for square matrices.
Inverse

Inverse relation

AX=B => X=A^-1B when A^-1 exists

Left multiply by inverse to solve.

  • Example: A^-1AX=A^-1B gives X=A^-1B.
  • Trap: Do not change order to BA^-1.

Derivation bank

Proofs that can be reproduced in exams.

Open a proof, follow the steps, and check the final result.

Prove (A+B)^T = A^T + B^T
  1. Let C=A+B, so c_ij=a_ij+b_ij.
  2. Entry ij of C^T is c_ji.
  3. c_ji=a_ji+b_ji.
  4. This equals entry ij of A^T+B^T.

Final: Transpose distributes over addition.

Shortcut: Use entry-wise equality.

Warning: A and B must have the same order.

Prove (AB)^T = B^T A^T
  1. Entry ij of (AB)^T is entry ji of AB.
  2. That equals sum a_jk b_ki.
  3. Entry ij of B^T A^T is sum (B^T)_ik (A^T)_kj.
  4. This equals sum b_ki a_jk, same scalar sum.

Final: Order reverses in transpose of product.

Shortcut: Compare ij entry.

Warning: Do not keep the order unchanged.

Show diagonal entries of skew-symmetric matrix are zero
  1. For skew-symmetric A, A^T=-A.
  2. Compare diagonal entry i,i.
  3. a_ii=-a_ii.
  4. So 2a_ii=0 and a_ii=0.

Final: a_ii=0 for all i.

Shortcut: Use A^T=-A on diagonal.

Warning: This assumes real entries.

Derive symmetric plus skew-symmetric decomposition
  1. Let S=(A+A^T)/2 and K=(A-A^T)/2.
  2. S^T=(A^T+A)/2=S, so S is symmetric.
  3. K^T=(A^T-A)/2=-K, so K is skew-symmetric.
  4. S+K=A.

Final: A=(A+A^T)/2+(A-A^T)/2.

Shortcut: Add the two parts.

Warning: Only square matrices can be decomposed this way.

Practice bank

Exam-style questions with answer reveal.

Filter by keyword, try the question first, then reveal the answer and trap.

CBSEEasy

1. What is the order of a matrix with 3 rows and 4 columns?

Answer: 3 x 4

Order is rows x columns.

Trap: Do not write 4 x 3.

CBSEEasy

2. Can a 2 x 3 matrix be added to a 3 x 2 matrix?

Answer: No

Addition requires same order.

Trap: Same number of entries is not enough.

ProductMedium

3. If A is 2 x 3 and B is 3 x 4, what is the order of AB?

Answer: 2 x 4

Inner dimensions 3 and 3 match; outer dimensions give 2 x 4.

Trap: Do not include the inner dimension in the result.

ProductMedium

4. Is AB always equal to BA?

Answer: No

Matrix multiplication is not generally commutative.

Trap: Even if both exist, they may differ.

TransposeMedium

5. Write transpose property for product AB.

Answer: (AB)^T = B^T A^T

Transpose reverses product order.

Trap: Do not write A^T B^T.

SpecialMedium

6. If A is skew-symmetric, what are its diagonal entries?

Answer: All are zero

a_ii=-a_ii gives a_ii=0.

Trap: This is for skew-symmetric matrices, not symmetric matrices.

InverseMedium

7. When does a square matrix have inverse?

Answer: When it is non-singular

For determinant chapters, this means determinant is non-zero.

Trap: Square condition alone is not enough.

JEEHard

8. If A is square, split A into symmetric and skew-symmetric parts.

Answer: (A+A^T)/2 and (A-A^T)/2

First part is symmetric, second is skew-symmetric, and their sum is A.

Trap: Do not forget division by 2.

CBSEMedium

9. What is AI when I has suitable order?

Answer: A

Identity matrix leaves A unchanged.

Trap: The order of I must be suitable.

ProductHard

10. If A is 3 x 2 and B is 2 x 3, what are orders of AB and BA?

Answer: AB is 3 x 3, BA is 2 x 2

Use outer dimensions each time.

Trap: AB and BA can both exist with different orders.

ProofHard

11. Prove (A+B)^T=A^T+B^T.

Answer: Compare entries

Entry ij on both sides is a_ji+b_ji.

Trap: Entry notation must be clear.

InverseMedium

12. Solve AX=B using inverse if A^-1 exists.

Answer: X=A^-1B

Left multiply both sides by A^-1.

Trap: Do not right multiply because order matters.

Core formulas

  • Addition rule: (A+B)_ij = a_ij + b_ij

  • Rows come before columns.
